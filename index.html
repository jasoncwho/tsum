Setup & Installation Guide

Click here to download the heart sender files

Follow these steps to configure your environment for the HeartSender automation script.

1. Install Python

  • Download the installer from python.org.
  • CRITICAL: During installation, check the box that says "Add Python to PATH".
  • Verify the installation by opening a terminal and typing:
    python --version

2. Enable Emulator ADB Connection

Open your emulator settings and ensure the ADB Debugging (ADB 偵錯) is set to "Enable local connection" (開啟本地連接) as shown below:

Reference image: adb.png

3. Install Python Libraries

Run these commands in your terminal/command prompt to install the necessary dependencies:

OpenCV (Image Processing):

pip install opencv-python

NumPy (Math & Arrays):

pip install numpy

PPADB (ADB Integration):

pip install pure-python-adb

4. Project File Structure

Ensure all files are in the same folder for the script to run correctly:

  • heart_sender.py
  • executor.py
  • Images: heart.PNG, ok_button.PNG, coins.png, close.png, close_cn.png, zero.PNG

5. Running the Program

Once the emulator is running and your game is at the ranking page , execute:

python heart_sender.py

The script will first receive present , and then scroll to the top to start sending hearts one by one.
